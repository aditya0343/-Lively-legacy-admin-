How it works

1. Resident Raises Complaint

Resident app saves complaint into complaints collection.

2. Admin Reviews Complaint

Admin checks priority, property and issue details.

3. Assign Staff

Assign available staff from the same property.

4. Track Progress

Update status as assigned, in progress, resolved or closed.

Quick Tips

  • Resident app and admin website must use the same complaints collection.
  • Staff must have same propertyId or propertyName to appear for assignment.
  • Use priority to handle urgent complaints first.
